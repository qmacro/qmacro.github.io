Series of "Did You Know" videos

  • 06 February 2026
  • dyk,
  • seriespost

Here's a collection of pointers to short "Did You Know" videos from the Developer Advocates, highlighting small but interesting and useful aspects of various SAP tech tools, frameworks and products.

In the run up to SAP TechEd 2025, the Developer Advocates put together a series of short videos on tech topics, in a "Did You Know" series. A bit like our collection of YouTube Shorts "Good To Know" series (on CAP Node.js and on ABAP Cloud), but in landscape mode with a bit more detail.

The series

Here's an overview of that "Did You Know" series; this list will be updated over the coming weeks as we add them one by one to upcoming SAP Developer News episodes. Use the 📺 link to jump directly to the section in the news episode.

1 The cds REPL

📺 Did you know: that the cds REPL is great for exploring your CAP services? More info: A reCAP intro to the cds REPL (SAP Developer News 29 Jan 2026, with DJ).

2 Mocking auth in CAP

📺 Did you know: that CAP has a "mocked" authentication strategy you can use in local development so you don't have to put off this important part of your domain design? More info: Authentication Strategies (SAP Developer News 05 Feb 2026, with DJ).

3 Debugging apps on CF

📺 Did you know: that it's easy to debug your CAP Node.js apps running on Cloud Foundry? More info: The Debugging with cds debug section of the CLI topic in Capire (SAP Developer News 12 Feb 2026, with DJ).

4 Released Objects in ADT

📺 Did you know: that the released objects that are available for developer extensibility are displayed in the Project Explorer of ABAP Development Tools? More info: The Released Development Objects section of the Developer Extensibility documentation in the SAP Help Portal (SAP Developer News 19 Feb 2026, with Shilpa).

5 Search ABAP CDS objects using Annotations

📺 Did you know: that you can search the CDS View entity by the annotations used? More info: The guide to the ABAP Development Tools for Eclipse in the SAP Help Portal (SAP Developer News 26 Feb 2026, with Shilpa).

6 Privileged access in ADT

📺 Did you know: that you can check if your data model can access the relevant data without using an access control? More info: The guide to the ABAP Development Tools for Eclipse in the SAP Help Portal (SAP Developer News 05 Mar 2026, with Shilpa).

7 AI Launchpad Model Library

📺 Did you know: that you can use the Model Library in Generative AI Hub on SAP AI Launchpad to explore the LLMs SAP is offering managed, hosted or provided remotely? More info: The SAP AI Launchpad Model Library documentation in the SAP Help Portal (SAP Developer News 12 Mar 2026, with Kevin).

8 Orchestration Client in SAP Cloud SDK for AI

📺 Did you know: that you can use the orchestration client from the SAP Cloud SDK for AI to create a full AI orchestration flow in CAP? More info: The SAP Cloud SDK for AI – Chat Completion documentation (SAP Developer News 19 Mar 2026, with Kevin).

9 Orchestration Client configuration

📺 Did you know: that you can configure an AI orchestration in AI Launchpad and use it to configure the orchestration client from the SAP Cloud SDK for AI? More info: Orchestration Workflow AI in the SAP Help Portal (SAP Developer News 26 Mar 2026, with Kevin).

10 Test Orchestration configuration

📺 Did you know: that you can test the orchestration configuration directly in AI Launchpad against various LLMs? More info: Test Your Orchestration Workflow in the SAP Help Portal (SAP Developer News 02 Apr 2026, with Kevin).

11 AI agents (Generative AI Hub & LangGraph)

📺 Did you know: that you can build AI agents using LangGraph and Generative AI Hub? More info: CrewAI example connecting to SAP Generative AI Hub via LiteLLM (SAP Developer News 09 Apr 2026, with Nora).

12 AI Launchpad Leaderboard

📺 Did you know: that you can compare foundation models in AI Launchpad using the leaderboard? More info: The Model Library in the SAP Help Portal for SAP AI Launchpad topics (SAP Developer News 16 Apr 2026, with Nora).

13 Override default auth while using xsuaa (CAP)

📺 Did you know: that you can override the default authentication behaviour for CAP service endpoints using a feature flag? More info: Authentication in Production - Enforced by Default in Capire. (SAP Developer News 23 Apr 2026, with Shrini).

14 Global canvas in Build Apps

📺 Did you know: that you can define logic in the Global Canvas and trigger it from any point in your app? More info: SAP Build Apps - Trigger Event in the SAP Help Portal. (SAP Developer News 30 Apr 2026, with Shrini).

15 Prompt Registry in Generative AI Hub

📺 Did you know: that you can store your prompts in the prompt registry and easily reuse them later? More info: Prompt Registry documentation in the SAP Help Portal. (SAP Developer News 08 May 2026, with Nora). 👈 LATEST
